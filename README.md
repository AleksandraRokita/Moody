## ✨ Główne funkcje

- **Dynamiczny Dashboard**: Spersonalizowane codzienne prognozy, śledzenie fazy cyklu i rekomendacje artykułów na dany dzień.
- **Zaawansowane Śledzenie (Tracking)**: Formularze dostosowujące się do obecnej fazy cyklu i ewentualnych zdiagnozowanych schorzeń (np. endometrioza, tarczyca). Śledzenie nastroju, energii, jakości snu i poziomu bólu.
- **Centrum Edukacyjne i Wellness**: Baza wiedzy zawierająca artykuły o diecie, psychologii i ciele, a także dedykowane techniki relaksacyjne (np. medytacje, ćwiczenia oddechowe).
- **Teczka Medyczna**: Wirtualny segregator do zapisywania wyników badań (np. profil hormonalny, morfologia) oraz przypomnienia o regularnych wizytach u lekarza.
- **Sieć Wsparcia (Network)**: Możliwość bezpiecznego udostępniania wybranych danych (np. o zbliżającym się PMS) partnerowi lub przyjaciołom w celu budowania wzajemnego zrozumienia.

## 🛠 Technologie

- **Frontend**: React (Vite)
- **Stylizacja**: Tailwind CSS (niestandardowa estetyka Y2K z wyrazistymi, grubymi obramowaniami i twardymi cieniami)
- **Ikony**: Lucide React
- **Routing**: React Router DOM
- **Deployment**: Gotowe do wdrożenia na platformie Vercel

## 🚀 Jak uruchomić lokalnie

1. Sklonuj repozytorium:
   ```bash
   git clone https://github.com/Kuczek03/Moody.git
   ```
2. Wejdź do folderu `frontend`:
   ```bash
   cd Moody/frontend
   ```
3. Zainstaluj zależności:
   ```bash
   npm install
   ```
4. Uruchom serwer deweloperski:
   ```bash
   npm run dev
   ```
5. Otwórz w przeglądarce adres `http://localhost:5173`.

# MOODY 💖

**Moody** is a web application designed to support menstrual health, track mood, and build better communication with loved ones. With its unique and bold **Y2K** design aesthetic, Moody not only helps you understand your body but also makes daily health tracking an enjoyable experience.

## ✨ Key Features

- **Dynamic Dashboard**: Personalized daily forecasts, cycle phase tracking, and tailored article recommendations for the day.
- **Advanced Tracking**: Smart forms that adapt to your current cycle phase and any diagnosed conditions (e.g., endometriosis, thyroid issues). Track mood, energy, sleep quality, and pain levels.
- **Education & Wellness Hub**: A knowledge base featuring articles on diet, psychology, and the body, alongside dedicated relaxation techniques (e.g., meditations, breathing exercises).
- **Medical Folder**: A virtual binder for saving lab results (e.g., hormone profiles, blood tests) and getting reminders for regular doctor check-ups.
- **Support Network**: Safely share selected data (like an upcoming PMS phase) with a partner or friends to foster empathy and mutual understanding.

## 🛠 Tech Stack

- **Frontend**: React (built with Vite)
- **Styling**: Tailwind CSS (custom Y2K aesthetic with distinct, bold borders and solid shadows)
- **Icons**: Lucide React
- **Routing**: React Router DOM
- **Deployment**: Ready for automated deployment on Vercel

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Kuczek03/Moody.git
   ```
2. Navigate to the `frontend` directory:
   ```bash
   cd Moody/frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and go to `http://localhost:5173`.
